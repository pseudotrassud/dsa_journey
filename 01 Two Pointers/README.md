# Two Pointers

## Description

The Two Pointers pattern uses two indices to traverse a data structure efficiently. Depending on the problem, the pointers may move toward each other, in the same direction, or from opposite ends.

This pattern is commonly used to reduce nested-loop solutions from **O(n²)** to **O(n)**.

---

## Recognition Clues

- Sorted array
- Pair-based problems
- Palindrome checking
- In-place modifications
- Comparing elements from opposite ends
